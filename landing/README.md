# Macroni Landing Page

Marketing landing page for [Macroni](https://github.com/leftautomated/macroni) — AI-powered desktop automation.

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Motion** (Framer Motion)
- **Inter** + **Geist Pixel Square** fonts

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
